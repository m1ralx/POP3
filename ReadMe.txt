POP3 Client by Alexey Mironov
usage: main.py


Следуйте инструкциям при вводе логина, пароля и выборе домена (если вы оставите поле домена пустым, то
программа сгенерирует его самостоятельно. Если вы вводите полный адрес электронной почты, то можно
оставить поле домена пустым.



Список команд:
----connect - подключение к серверу и передача ему логина и пароля
----get arg1 [arg2] получение письма с номером arg1 или всех писем в диапазоне [arg1, arg2]
----delete arg удаление письма с номером arg
----quit завершение сеанса работы с POP3 (закрытие сокета)

Описание:
pop3.py -- модуль, содержащий класс POP3, который реализует работу с pop-сервером с помощью
команд протокола POP3
email_parser.py -- модуль, содержащий класс Parser, который реализует функциональность
обработки письма, полученного с использованием методов класса POP3.
